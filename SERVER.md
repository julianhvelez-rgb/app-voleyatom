# VolleyAtom - Servidor Local

Servidor HTTPS para habilitar Web Bluetooth API

## Iniciar servidor

```bash
node server.js
```

Luego abre: **https://localhost:3000**

## Web Bluetooth habilitado

✅ Conexión directa con impresora AGI-PR6000UBT-3748
✅ Certificados HTTPS auto-configurados
✅ Compatible con Chrome, Edge, Opera

## Nota

Si ves advertencia de seguridad en el navegador, es normal para certificados locales.
Haz clic en "Avanzado" > "Continuar a localhost"
